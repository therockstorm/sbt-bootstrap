# sbt-bootstrap

[sbt](https://www.scala-sbt.org/) plugins for common project settings.

* `Bootstrap`
  * `scalafmt` and `scalafix` config
  * git hook installation via [lefthook](https://github.com/Arkweid/lefthook)
  * Compiler flags
* `BootstrapIt`
  * Common integration test config

## Publish locally

```shell script
sbt publishLocal
```
